Project is created with:
* React JS
* React Bootstrap
* Sass
* [AOS Animation](https://github.com/michalsnik/aos)

## 💡 Key Concepts
* React Router
* Single Page website
* MVC (Model-View-Controller)

## 💻 Setup
To run this project, install it locally using npm:

```
$ cd ../newsmagazine
$ npm install
$ npm build
$ npm start
```
